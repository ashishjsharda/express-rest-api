# Express API Demo

This is a basic Express API demo that demonstrates how to create GET and POST endpoints using Express.js.

## Installation

1. Clone the repository:

   ```shell
   git clone https://github.com/your-username/express-api-demo.git

   cd express-api-demo
   
   npm install
   
   

    ```
2. Start the server:
   ```shell
   node src/server.js
   ```
